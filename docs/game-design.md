# もこもん ゲーム設計書

プロトタイプ(`prototype/mokomon.html`)で実証済みの仕様。**現在は Flutter 版(`app/`)とこのドキュメントが正**。プロトタイプは検証時点の凍結資料であり、以後の変更(種族8-9、進化ビジュアル強化など)は反映しない。

## 1. コンセプト

- ターゲット: 6〜7歳(+親)。文字はすべてひらがな・カタカナ
- コアループ: **ミニゲームでコインを稼ぐ → ごはんで育てる → 劇的に進化 → ずかんが埋まる → 新しいたまご**
- セッション設計: 1回5〜10分。お腹/ごきげんの時間減衰で「また開く理由」を作る

## 2. 状態(セーブデータ)

**しゃべり声(バブル音声)**: なでなで・ごはん後・ゲームから帰宅時などに、種族ごとの高さの「しゃべり風」合成音声(5〜8音節・3バリエーション)を再生する。どうぶつの森式の意味のないおしゃべり。
**ひとりごと**: 放置中、20〜32秒ごとに勝手にしゃべる(セリフ8種+バブル声+もじもじ)。たまご中は「たまごが ゆれた…!?」と揺れる。

**誇張表情(リアクション中だけ顔が差し替わる・全種族共通)**: なでなで/褒め=にっこり(∩∩目+大口+舌)、💨=びっくり(まん丸目+高眉+「お」の口)、ごはん=あーん(とじ目+大きく開けた口)。約1〜1.6秒で種族の顔に戻る。
**セリフの吹き出し**: 大きな文字(26px相当・自動縮小)+ぽよんと弾む登場+毎回わずかに傾く。

### 進化(隠しパラメータ)
- しきい値: stage1→2 は xp **30**、stage2→3 は xp **80**
- 予兆: しきい値の 8/12 手前から金色グロー+✨+一度だけ吹き出し「なんだか からだが ひかってる…!」
- **ゲージ等の数値表示は絶対にしない**(検証で「サプライズが死ぬ」と判明)
- キング到達で collection 登録、ずかんから新しいたまごを迎えられる

### 進化カットシーン(重要・検証で最も効果があった演出)
1. 全画面暗転(紺のラジアル)+白い光条が回転
2. 現形態が真っ白なシルエットで震える(2.4秒、振幅漸増)+上昇音階(180Hz→1000Hz強、13音)
3. 白フラッシュ(0.9秒)
4. 新形態リビール(ポップイン+金ドロップシャドウ)+紙吹雪+ファンファーレ+「〜に しんかした!!」
5. 「すごい!!」ボタンで閉じる

### 進化ごとの見た目(Flutter版で強化: シルエット単位で変える)
- **ベビー**: スケール0.62・専用の丸い素体・アクセサリ(耳/トゲ/星)なし・腕なし
- **中間**: 標準体型。種族アクセサリが生え、腕(両サイドの短い手)が生える
- **キング**: スケール1.2・王家のマント(紅+金縁)・おなかの明色パッチ・アクセサリ1.18倍・王冠、背後に金色オーラ(回転する放射、mask で円形フェード)

※プロトタイプは「ベビー0.7/キング1.14・アクセサリはstage1+」の簡易版。Flutter版が正。

## 5. ミニゲーム(10種)

共通報酬: クリアで happy+12、xp+10(+ゲーム固有コイン)

### フルーツキャッチ(反射)
30秒。0.45〜0.85秒間隔で落下物(🍎🍊🍇🍓🍌、14%で⭐)。落下速度120〜220px/s+横揺れ。
**残り時間が減るほど加速**(出現間隔・落下速度とも最大1.9倍。こどもFBで難化)。
タップ判定半径44px。フルーツ+1コイン、⭐+3コイン。開始前に3,2,1カウントダウン。

### おなじのどれ?(パズル)
8ラウンド。形(丸/星/三角/四角/ハート)×色(5色)のターゲットと同じものを4択から(こどもFBで難化)。
不正解はシェイクのみでペナルティなし(再挑戦可)。正解+2コイン、進捗ドット表示。

### ペアさがし(記憶)
4×5=20枚(10ペア: 🍎🍌🍇⭐🐟🌸🍩🐸🚗🌙)。カードフリップは3D回転0.35秒。
不一致は0.75秒後に伏せる。全ペアで+20コイン。(2人プレイでちょうどよい難易度に増量。こどもFB)

### もぐらたたき(反射・Flutter版で追加)
30秒。3×3の穴からいきもの(実際の種族絵)が0.75〜1.2秒だけ顔を出す。最大3匹同時。
ふつう+1コイン、きんいろ(ぴか)12%で+3、💨もぐら10%は0コイン(おふざけ)。終盤ほど加速(×1.8)。

### なぞってかこう(おえかき系・Flutter版で追加)
点線の形(ほし/はーと/まる/つき/かみなり)から3つをなぞる。輪郭カバー率で
★3(85%+, +4コイン)/★2(55%+, +3)/★1(+1)。おえかき好きのこども向け。

### ちがうのどっち?(観察・Flutter版で追加)
にている絵文字ペア(🍎🍅・😀😃・😺😸・🌕🌝等13組)のうち1つだけ違うものを探す。
8ラウンド、グリッドは12→16→20→25枚と拡大(こどもFBでさらに難化)。正解+2コイン。

### ふうせんわり(反射・Flutter版で追加)
30秒。下からふわふわ上がる🎈をタップ(+1)。⭐14%は+3、💣14%はさわると-2(0未満なし)。終盤加速(×1.8)。

### じゅんばんタッチ(かず・Flutter版で追加)
1〜9のボタンを順番にはやくタッチ。タイムでコイン(11秒未満+16 / 20秒未満+10 / それ以上+6)。

### かぞえてタッチ(かず・こどもFBで追加)
ちらばった絵文字(にたものが2種まざる)から対象をかぞえて3択で答える。
6ラウンド、9→24枚と増加。まちがえてもペナルティなし(数えなおし可)。正解+3コイン。

### おぼえてタッチ(記憶・こどもFBで追加)
4色パッドが光ったじゅんばんを覚えて同じ順にタッチ(サイモン)。2連から始まり
クリアごとに+1、最大7連。1ラウンド+3コイン、まちがえたらそこで終了(獲得分は持ち帰り)。

## 6. お絵かき(もようがえ)

- 体形のパスでクリップした透明レイヤーに描く。ツールは**ふで(ふとさ3段階)/スタンプ/バケツぬりつぶし/けしごむ**
- ブラシ**16色**。バケツは線の境界と体マスクで止まる flood fill(logic/flood_fill.dart)
- **おもしろスタンプ20種**+おみやげ解放の限定スタンプ(§14): タップでペタッ、なぞると連続でペタペタ
- パレット類は画面下部に配置(手のひら誤タップ対策)。キャンバスは余白なしで最大化(最大400px)
- **UI方針(こどもFB)**: ダイアログはスクロールさせない。ごはん(10種)/あそぶ(10種)は2列グリッド、ずかんは4列(セル大きめ)、おみせは「あたま/かお/はいけい」タブ
- **モーダルはレスポンシブ**(`MokoModalShell`): 見出し・操作ボタンは画面サイズによらず常に見える位置に固定し、本文(グリッド等)だけを画面高さに応じて必要なときにスクロールさせる。端末が小さくても「とじる」「むかえる」等のボタンに毎回たどり着ける
- 体の色も6色から変更可
- 保存した模様は体に合成表示。**体色とレイヤーを分離**(後から体色だけ変えられる)
- happy+8、xp+4
- 保存してホームに戻ると、いきものが褒めてくれる(スピン1回転+キラキラ6個+褒めセリフ4種からランダム。Flutter版で追加)

## 8. あいことばセーブ(パスワード方式)

- 形式: `MOKO-` + base64(`ver,stage,xp,coins,hunger,happy,species,collectionBits,ownedBits,equipHeadIdx,equipFaceIdx;checksum`)
- checksum: body全文字コード和 mod 97
- 模様(pattern)は含めない(サイズ都合)。復元時は pattern=null, eggTaps=0
- 復元時は範囲クランプ+未所持アイテムの装備解除。エラー時は絵文字入りのやさしい文言

## 9. 隠し機能「💨」(名前を付けないこと)

- 発動: いきものの下部30%をタップ、または通常なでなでの6%で暴発
- 演出: 低い下降音6連「ブゥ〜ブブブッ」(155→66Hz・うなり付き)+💨🌫️が**5つ大きく**横に流れて消える+体よじり+びっくり顔(こどもFBでおなら感アップ)
- 吹き出し(ランダム6種): 「……いまの なあに?」「あれ? へんな おとが した!」「……なにも してないよ?」「ぷぅ♪」「きこえなかった ことに しよう…」「だ、だれかな? いまのは…」
- **UI上にボタンや説明を一切出さない**(発見型コンテンツ)

## 10. サウンド

全てWebAudio(Flutter版は実行時にWAVを合成・アセット不要)。
tap/coin/munch/happy/wrong/pop/fanfare/evolution riser/puff、鳴き声 coo/giggle、
種族ごとのしゃべり声(バブル音声・決定的生成)。

**派手ジングル(Flutter版で追加)**: 鳴っている間はBGMを一時停止して主役にする。
- megaFanfare: たまご孵化の瞬間(駆け上がり+和音+シャンシャン。進化リビールと同じ路線)
- rewardJingle: ミニゲーム報酬(タタタ・ジャーン!+キラキラ)
- dressUp: きせかえ購入(ハープグリッサンド+ベル)。装備切替は控えめ版

**BGM**(🎵ボタンで切替・選択保存): ホーム3曲(そよかぜ/わくわく/ぽかぽか)+**ゲームBGM2曲**(140bpm疾走/120bpmポップ・遊ぶたびランダム)+**おえかき専用まったり曲**(84bpmアルペジオ)。場面ごとに自動で切り替わる。
**勝利曲(victoryTune・約9秒)**: ミニゲームで**20コイン以上のすごいスコア**のとき、および**進化リビール**で流れる。終わると元のBGMへ自動復帰。進化カットシーンの震え中はBGM停止、バックグラウンドで自動停止・復帰で再開。
ミュートトグルあり(設定保存・BGM連動)。

## 14. キングのおみやげ(Flutter版で追加)

- キング(stage3)はメーターに**✨きらきらゲージ**(0-100)が追加される
- なでなで+10 / 💨+6 / ごはん+16 / ミニゲーム+30 / おえかき保存+25 で溜まる(なでなで10回で満タン)
- 50%と85%で吹き出し(「きらきらが たまってきた…!」等)を出し、進捗の手ごたえを見せる
- 満タンで🎆おみやげ: 最初の3回は**限定スタンプ解放**(👑→🎆→🦄、+10コイン付き)、以降はコイン20〜40
- ゲージは個体ごと(名簿に保存)。スタンプ解放は端末共通。あいことばには含めない

## 13. 背景テーマ(Flutter版で追加)

- **19テーマ**: 最初の11種(そら☁️/ゆうやけ🌇/よぞら🌙/うみ🐠/もり🌳/ゆき⛄/うちゅう🪐/さばく🌵/ゆうえんち🎡/かざん🌋/にじぞら🌈)は**無料**。
  追加の8種(こどもFBで追加)は**コインで購入**: ふかいうみ🐋25 / ジャングル🐒20 / キャンディランド🍬20 / おしろ🏰30 / オーロラ🌌30 / おんせん♨️20 / よいちまつり🏮25 / はなばたけ🌷15
- **色はガラッと変える**(こどもFB): よぞら・うちゅうはほぼ真っ暗+星、ゆきはほぼ真っ白+雪、かざんは暗赤+火の粉。テーマ別の飾りも多め
- **種族ごとにデフォルト背景**(`data/backgrounds.dart` の speciesDefaultBg。例: おばけん→よぞら、ろぼっち→ゆき)は必ず無料テーマの中から選ぶ。種族追加時は必ず1エントリ足す
- ショップの「はいけい」セクションから切替。未購入の有料テーマをタップするとコインを消費して購入(即選択)、購入ずみなら無料で選び直せる。「おまかせ🎲」でデフォルトに戻す(常に無料)
- 選択(bg)は個体ごとに保存(名簿にも)。購入状況(ownedBg)は端末で共通・全個体で共有。いずれもあいことばには含めない(**追加は必ず末尾**)

## 12. 名簿(roster)と交代・ニックネーム(Flutter版で追加)

- キングを育てて新しいたまごを迎えると、その子は**名簿**に保存される(stage・体色・模様・きせかえ・なまえ)
- **ずかんの入手済みセルをタップで交代**できる。いま育てている子(たまご・育成途中でも)は名簿に退避され、続きから再開できる
- きせかえの**装備状態は個体ごと**に保持(所持アイテム自体は共有)
- **ニックネーム**: 下部カードの名前ピルをタップ→10文字まで。個体ごとに保存。名簿・ニックネームは端末ローカルのみで、あいことばには含めない(復元時はリセット)
